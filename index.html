EONTZIA

eOntzia es un sistema inalámbrico que mide la capacidad de los contenedores de residuos. Es versátil y modular, es decir, se puede moldear a gusto de cada cliente y se adecua a sus necesidades. Como ventaja principal, cabe destacar su bajo coste y su movilidad o adaptabilidad ya que es ajustable en cualquier tipo de contenedor existente.

Con este sistema se pretende mapear y ubicar todos los contenedores de residuos en los que se interese controlar el estado de llenado, ya sea en grandes ciudades, o bien, en pequeños municipios alejados en zonas rurales. En la aplicación web se podrá gestionar tanto los trabajadores que tiene la empresa como el estado de cada uno de los contnedores plasmados gráficamente en un mapa. También dispondrá de gráficos visuales de los contenedores.

Peticiones API eOntzia:
La aplicación dispone de un apartado API a las cuales se le pueden consultar los datos. Las directrices para realizar dichas peticiones son las siguientes:
La url base será: http://eontzia.zubirimanteoweb.com/app

-Realizar una lectura del contenedor:

Petición tipo GET
URL: /nuevaLectura/[Id dispositivo]/[Volumen]/[fuego]/[bateria]
Id_dispositivo → de momento hemos metido en la base de datos dos dispositivos que tienen el id 1 y 2.
Volumen → numero entero de 0 a 100.
Fuego → 1 --> hay fuego, 0 --> no hay fuego
Bateria→ numero entero de 0 a 100.

Ejemplo: http://eontzia.zubirimanteoweb.com/app/nuevaLectura/1/55/0/99

-Realizar una lectura de datos guardados en el servidor:

URL: /getLecturas/[id dispositivo]
Ejemplo: eontzia.zubirimanteoweb.com/app/getLecturas/1
